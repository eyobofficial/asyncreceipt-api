# Receipt Generator API
A Python/Django RESTful API for generating receipts for authenticated and authorized users.

## Requirements
- Python 3.8
- Pipenv
- Redis

## Packages
- Django 3.1.7
- Django REST Framework 3.12.2
- Gunicorn
- Celery
- Python Decouple
- Django Environ
- Pendulum

## DevOps Tools
- Travis CI
- Docker
